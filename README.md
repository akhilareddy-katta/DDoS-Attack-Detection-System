# DDoS Attack Detection System

## Project Overview

This project is a Deep Learning based DDoS Attack Detection System developed using TensorFlow, Python, and Streamlit. The system analyzes network traffic data and classifies it as either Normal Traffic or DDoS Attack.

## Features

* Upload network traffic CSV files
* Detect DDoS attacks automatically
* Deep Learning based classification
* Streamlit web dashboard
* Uses CICIDS2017 dataset

## Technologies Used

* Python
* TensorFlow
* Scikit-Learn
* Pandas
* NumPy
* Streamlit

## Dataset

CICIDS2017 Dataset

## How to Run

### Train the Model

```bash
python train_model.py
```

### Run Dashboard

```bash
streamlit run dashboard.py
```
